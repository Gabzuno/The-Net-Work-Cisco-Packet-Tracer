Bandwidth - is the capacity of a medium to carry data.

Throughput - is the measure of the transfer of bits across the media over a given period of time.

Latency - refers to the amount of time, including delays, for data to travel from one given point to another.

Computer - an electronic system that operated by commands stored in its own memory. It can accepts data(inputs), process data through specified rules, produce information (output), and store for future use.

Characcteristics : 

&nbsp;   Speed - can process data at the rate of millions of instructions per second.

&nbsp;   Accuracy - offers high degree of precision.

&nbsp;   Diligence - doesn't get tired

&nbsp;   Storage Capability - store large quantities of data and information.

&nbsp;   Versatility - naturally flexible. Can perform various task/activities.

Viruses - is a very small malicious software that hides in real programs.

&nbsp;   E-mail Viruses - spreads to e-mail messages as an attachment.

&nbsp;   Trojan Horses - a program for computer. Claims to do one thing, but when executed it does harm instead.

&nbsp;   Worms - a tiny piece of software that replicates itself using computer networks and security holes. It uses the security holes replicating itself from to the new computer.
